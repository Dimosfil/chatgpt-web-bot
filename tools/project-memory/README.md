# Project Memory — chatgpt-web-bot

Эта папка хранит долговременные знания о проекте для AI-агентов.

## Назначение

Факты, которые стоит записать здесь (не только в чате):
- архитектурные решения и их обоснования
- результаты отладки сложных проблем
- важные конфигурационные нюансы
- частые ошибки и их решения
- полезные команды и их наблюдаемое поведение

Не хранить: секреты, токены, пароли, продакшен-данные.

## Файлы

- `NOTES.md` — заметки, не вошедшие в другие файлы
- `known-issues.md` — известные проблемы и workaround'ы

## Правило

Если будущий агент потратит время, переоткрывая тот же факт — запиши его сюда.
